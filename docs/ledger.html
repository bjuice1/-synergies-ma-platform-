System Overview

Architecture

Synergies is an M&A synergy analysis SaaS platform for deal teams. Analysts input deals, extract company data from briefing documents via Claude, benchmark the deal against comparable transactions (APQC dataset), and track lever-level value capture through a structured diligence workflow.

Scale Readiness 6/10
✅ Works for 2–10 analysts, <50 deals  ·  ⚠️ Needs work above 100 concurrent users
Tech Debt 5/10
🟠 Fake benchmark data (primary risk)  ·  🟡 Manual schema migrations
Code Complexity 6/10
🟢 Clean Flask/React patterns  ·  🟡 Benchmark data model non-trivial
Security 7/10
🟢 JWT gated, role-checked  ·  🟡 /api/learn is intentionally public

Schema Migrations

⚠️
Critical: db.create_all() in releaseCommand only creates NEW tables. It silently ignores existing tables and will NOT run ALTER TABLE. All column additions must be applied manually.
# Connect directly to Railway Postgres
psql "postgresql://postgres:<password>@maglev.proxy.rlwy.net:47787/railway"

# Add new column (safe — IF NOT EXISTS prevents errors on re-run)
ALTER TABLE deal_levers
  ADD COLUMN IF NOT EXISTS my_new_col TEXT;

# Verify
SELECT column_name, data_type
FROM information_schema.columns
WHERE table_name = 'deal_levers'
ORDER BY ordinal_position;
